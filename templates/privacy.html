Privacy & Data Notice

How MAIP handles your uploaded files and session data

✓ No account required · No permanent storage · No data sharing

What MAIP collects

When you use MAIP, the following information is processed:

  • Uploaded file — the .xlsx, .xls, or .csv file you choose to upload
  • Session ID — a randomly generated identifier that links your queries to your uploaded data
  • Query text — the natural language questions you type into the chat interface
  • IP address — recorded in the audit log for rate limiting and abuse prevention
Your uploaded file is never shared with other users. Each session is fully isolated. Two people using MAIP at the same time cannot see each other's data.

How long data is stored

Uploaded file

Held in your session for up to 4 hours, then automatically deleted.

Query history

Stored in the session log for 4 hours, then deleted with the session.

Research results

Cached for performance during your session. Deleted when the session expires.

IP address

Retained in the audit log for 30 days for rate limiting purposes only.

How to delete your data immediately

Close the browser tab. Your session expires and all associated data is scheduled for deletion. There is no account to log out of and no data to manually request deletion of — the session lifecycle handles it automatically.

Third-party processing — Anthropic API

MAIP uses the Anthropic Claude API to process your natural language queries. When you type a question, the following is sent to Anthropic:

  • Your query text (after abbreviation expansion)
  • Pre-computed statistics about your filtered data (counts, breakdowns — never raw rows)

Raw data rows from your uploaded file are never sent to Anthropic. The AI only receives aggregate statistics that have already been computed by the server.

Anthropic's data handling is governed by their Privacy Policy. MAIP uses the API under Anthropic's standard commercial terms.

Market intelligence research

When you use modules such as Competitive Landscape or HTA Profiles, MAIP sends a research query to the Anthropic API with web search enabled. These queries contain only the product name, country, and therapy area you selected — never any data from your uploaded file.

What MAIP does not do

  • Does not store your uploaded file permanently
  • Does not share your data with other users or third parties (except Anthropic API as described above)
  • Does not require account creation or login
  • Does not use cookies for tracking
  • Does not send your raw data rows to any AI model
  • Does not retain data after your session expires

Security

MAIP is served over HTTPS. The Anthropic API key is stored server-side only and is never exposed to the browser. File uploads are validated for type and size before processing. Rate limiting is applied per IP address to prevent abuse.

Children

MAIP is a professional tool intended for use by adults in pharmaceutical market access roles. It is not directed at children under 13.
